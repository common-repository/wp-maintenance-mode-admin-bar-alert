== Description ==

Checks whether the Maintenance Mode of the [WP Maintenance Mode][]
plugin is on and, if so, adds an alert to that effect in your Admin
Bar.

[WP Maintenance Mode]: http://wordpress.org/extend/plugins/wp-maintenance-mode/


Note that WP Maintenance Mode includes an admin bar alert as of
version 1.6.0, making this plugin redundant for those who use that
version or above.


== Installation ==

1. Upload the `wp-maintenance-mode-admin-bar-alert` folder to the `/wp-content/plugins' directory
2. Activate the plugin through the 'Plugins' menu
